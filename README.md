# WayPoint Mental Health Support App

This is a mental health support application developed using React with TypeScript and Vite. The app provides a platform with various pages such as Booking, Chatbot, Forum, Landing, and Resources.

## Project Structure

```
WaypointDemo/
├── src/
│   ├── components/
│   │   ├── Footer.tsx    
│   │   ├── Navbar.tsx    
│   │   └── Sidebar.tsx   
│   ├── pages/
│   │   ├── Booking.tsx   
│   │   ├── Chatbot.tsx   
│   │   ├── Forum.tsx     
│   │   ├── Landing.tsx   
│   │   └── Resources.tsx 
│   ├── App.tsx           
│   ├── index.css         
│   ├── main.tsx          
│   └── vite-env.d.ts     
├── index.html            
└── package.json          
```

## Prerequisites

- [Node.js](https://nodejs.org/) (version 14 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) as your package manager

## Installation

1. **Clone the Repository:**

   ```sh
   git clone <repository-url>
   cd WaypointDemo
   ```

2. **Install Dependencies:**

   Using npm:

   ```sh
   npm install
   ```

   Or with yarn:

   ```sh
   yarn
   ```

## Running the Application

Start the development server by running:

Using npm:

```sh
npm run dev
```

Or with yarn:

```sh
yarn dev
```

Open the provided URL in your browser (commonly http://localhost:3000 or http://localhost:5173) to view the app.

## Building for Production

To build the project for production, run:

Using npm:

```sh
npm run build
```

Or with yarn:

```sh
yarn build
```

## Additional Information

- The application is scaffolded with Vite.
- For additional project-specific scripts and configurations, refer to the `package.json` and `vite.config.ts` files.
